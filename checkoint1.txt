 How does the web work?
	-The Web is a massive network of webpages, programs, and files that are accessible via URLs.
	- Web protocols:The World Wide Web. Domain Name System (DNS) Practice: Hypertext Transfer Protocol (HTTP) Practice
	-The web browser connects to the web server and sends an HTTP request (via the protocol stack) for the desired web page.
	-The web server receives the request and checks for the desired page. If the page exists, the web server sends it.
	 If the server cannot find the requested page, it will send an HTTP 404 error message.
